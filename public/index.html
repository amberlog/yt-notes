1. Download subtitles

Pull subtitles from YouTube to local disk. Two input modes.

Enter a YT channel + start date. Grabs all English subs for videos longer than 5 min.

Paste a single video URL to grab its subtitles.

2. Pick transcripts

Select transcripts to send to the LLM (multi-select).

Once subtitles are downloaded, transcript files appear here.

3. Summarize with LLM

Use DeepSeek to turn selected transcripts into markdown notes.

First prompt you type is auto-saved as default

4. Result & save

Download .md or save to a group.
